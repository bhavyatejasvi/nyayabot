# NyayaBot (skeleton)

This repository contains a minimal runnable skeleton of the NyayaBot WhatsApp webhook server described in CLAUDE.md. It sets up an Express server with a `/webhook` endpoint and basic configuration. Use this as a starting point to implement full modules and integrations.

Quick start

```powershell
cd c:\Users\Dell\Desktop\nayayabot
npm install
npm run dev
```

Fill in `.env` values from `.env.example` before connecting Twilio/Anthropic/OpenAI/Supabase.
